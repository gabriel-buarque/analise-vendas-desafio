
📊 Análise de Dados de Vendas com Python e Pandas

Projeto de análise exploratória de um dataset de vendas utilizando Python, Pandas e Matplotlib.

🎯 Objetivo

Analisar um conjunto de dados de vendas para descobrir:

- 🏆 Produto mais vendido
- 💰 Total de vendas (unidades e valor em R$)
- 📅 Vendas por data
- 📈 Visualização com gráficos

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

📁 Estrutura do Projeto


📦 analise-vendas
 ┣ 📓 desafio_vendas.ipynb   # Notebook com a análise completa
 ┗ 📊 vendas_desafio.xlsx    # Dataset com 1000 registros de vendas


🚀 Como Executar

1. Clone o repositório
```bash
git clone https://github.com/gabriel-buarque/analise-vendas.git

2. Instale as dependências
```bash
pip install pandas matplotlib openpyxl
```

3. Abra o notebook no VSCode e execute as células
